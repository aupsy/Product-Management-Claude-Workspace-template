# User Research

User interviews, personas, usability findings, and Jobs-to-be-Done analysis.

## What Belongs Here

- User interview notes and summaries
- Persona documents
- Usability test findings
- Jobs-to-be-Done (JTBD) analysis
- NPS / CSAT analysis write-ups
- Field research or observation notes

## Tips

- Claude uses this context when writing PRDs — add user research here and ask Claude to incorporate it
- Link research to specific features: "Research supporting [Feature X]: `user-research/feature-x-interviews-20260301.md`"
- Even a short summary of 3 user calls is more useful than nothing
