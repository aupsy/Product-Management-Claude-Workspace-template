## How to Use This

For each strategic objective, define:
- **Metric**: A specific, measurable number that proves you've achieved the objective
- **Target**: The number you're aiming for and by when
- **Current**: Where you are today
- **Tactics**: The specific actions that move the metric

The strongest SMTs have metrics that are:
- **Specific** — no ambiguity about what's being measured
- **Causal** — the metric actually reflects the strategy, not a proxy
- **Actionable** — you know what to do differently if the number is wrong
