## The SMT Framework

Every metric should trace back to a strategic objective. The SMT (Strategy → Metrics → Tactics) framework ensures you're measuring the right things:

- **Strategy**: What outcome are you trying to achieve?
- **Metrics**: How will you know you've achieved it?
- **Tactics**: What actions move those metrics?

See `SMT-framework-template.md` for the full template.
